## 🏁 Getting Started <a name = "getting_started"></a>

Clone

```
git clone https://github.com/raferdev/ignite_call
```

Install Dependencies

```
npm i
```

Start Server

```
npm run dev
```

Make requests to the endpoints.

```
http://localhost:3000
```
